Concept - display for La Marzocca Linea Micra to pull data realtime via Home Assistant connection. Screen shows boiler temp, steam temp, estimated shots left, wifi connectivity. It wakes the screen on device power on and dims at other times after 30 seconds. Holding down both buttons resets the drink per tank estimator.

1) Used a Lilygo T-Display TTGO
2) Install ESPHome on Home Assistant
3) Install File Editor add-on to upload icon file and put in /homeassistant/esphome/images
4) Create secrets file
5) Install ESPHome via the browser to get the working BIN file
6) Flash to device via USB-C cable from PC
